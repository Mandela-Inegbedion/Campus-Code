# Django Girls Image Prompts

These prompts are prepared for the Django Girls landing page at `django-girls/index.html`.

Recommended output style for all assets:
- Professional website-quality raster images
- Warm, welcoming, community-first tone
- Natural lighting where applicable
- No watermarks
- No visible copyrighted logos unless explicitly requested
- No in-image text unless explicitly requested

## 1. Hero Workshop Banner

Filename:
`hero-workshop-banner.png`

Prompt:
```text
Use case: photorealistic-natural
Asset type: landing page hero image
Primary request: create a warm, welcoming workshop scene showing women learning programming together in a beginner-friendly coding event
Scene/backdrop: modern workshop space with tables, laptops, notebooks, soft daylight, community atmosphere
Subject: diverse women in tech participating in a coding workshop, smiling, collaborating, one person helping another
Style/medium: editorial-style realistic photography
Composition/framing: wide composition suitable for website hero, strong focal group on the left-center with some breathing room for page layout
Lighting/mood: bright, natural, inspiring, supportive, inclusive
Color palette: warm neutrals with gentle orange and gold accents
Materials/textures: real laptops, tables, notebooks, casual clothing, realistic indoor textures
Constraints: no text, no watermark, no visible brand logos, no exaggerated AI artifacts
Avoid: empty room, corporate stiffness, dark moody lighting, overly staged stock-photo look
```
